# Instruções de Uso por Tela

Este guia explica como usar cada tela do aplicativo **Mensagem Transformadora**. O app funciona **100% offline**: tudo é salvo localmente no seu dispositivo.

---

## 1) Início

**Objetivo**: acesso rápido às principais ações e às últimas mensagens criadas.

**Como usar**
- Toque em **Nova mensagem** para criar uma nova anotação de pregação.
- Toque em **Ver histórico** para acessar a lista completa.
- Em **Últimas mensagens**, toque em qualquer card para abrir os **Detalhes**.
- No ícone de **informação (i)** no topo, acesse a tela **Sobre o Aplicativo**.

---

## 2) Histórico

**Objetivo**: ver todas as mensagens cadastradas e encontrar rapidamente uma anotação antiga.

**Como usar**
- Digite no campo de busca para filtrar por:
  - título
  - pregador
  - igreja
  - versículo base
  - versículos secundários
- Toque em um card para abrir os **Detalhes**.
- Toque no botão **+** (flutuante) para criar uma **Nova mensagem**.
- Toque no ícone de **Início** no topo para voltar à tela inicial.

---

## 3) Nova Mensagem

**Objetivo**: cadastrar uma nova anotação de pregação.

**Campos obrigatórios**
- Nome do usuário
- Nome do pregador
- Igreja
- Data (automática na criação)
- Título da pregação
- Versículo base

**Como usar**
- Preencha os campos obrigatórios.
- Use **+ Adicionar** para inserir **versículos secundários** (remova com o ícone de lixeira).
- Use **+ Adicionar** para inserir **pontos principais** (remova com o ícone de lixeira).
- Em **Frases marcantes**, escreva **uma frase por linha**.
- Toque em **Salvar**. Você será levado automaticamente para os **Detalhes** da mensagem recém-criada.

---

## 4) Detalhes

**Objetivo**: visualizar a mensagem completa e executar ações.

**Ações no topo**
- **Estrela**: favoritar/desfavoritar a mensagem.
- **Duplicar**: cria uma cópia (útil para reutilizar um “modelo”).
- **Excluir**: remove a mensagem (com confirmação).
- **Editar**: abre o formulário de edição.

**Como usar**
- Role para ver as seções (versículos, pontos principais, frases marcantes, anotações e resumo final).
- Toque em **Voltar** para retornar à tela anterior.

---

## 5) Editar

**Objetivo**: atualizar uma mensagem existente.

**Como usar**
- Ajuste qualquer campo necessário (inclusive listas dinâmicas).
- Toque em **Salvar alterações** para atualizar o registro local.
- Ao salvar, o app retorna para os **Detalhes**.

---

## 6) Dashboard

**Objetivo**: acompanhar estatísticas de uso do app.

**O que você verá**
- Total de mensagens cadastradas.
- Total de favoritos e proporção.
- Gráfico de volume dos últimos 7 dias.
- Ranking de pregadores e igrejas mais frequentes.

**Como usar**
- Entre na aba **Dashboard** para atualizar as métricas automaticamente.
- Use o botão de **atualizar** (ícone de refresh) para recalcular os números.

---

## 7) Guia (Instruções)

**Objetivo**: consultar as orientações de uso dentro do próprio aplicativo.

**Como usar**
- Abra a aba **Guia** no rodapé.
- Leia as instruções por tela, com destaques de pontos importantes.

---

## 8) Sobre o Aplicativo

**Objetivo**: ver informações do desenvolvedor e versão.

**Como usar**
- Acesse pelo ícone **(i)** na tela Início.
- Consulte versão, autor, empresa e licença.
